Laminas\Http\PhpEnvironment\Request 
------------------------------------
- getMethod()                  | Return the method for this request. (return string) -->JInput::getMethod()
- isOptions()                  | Is this an OPTIONS method request? (return bool)
- getHeader()->getFieldValue() | Returns a https request header value based on the provided name. (return string) --> JInput->server->get('HTTP_CONTENT_TYPE')
- getUri()->getPath()          | Returns the URI path. (return string)  --> JUri::getPath()
- getQuery('info', -1)         | Returns a single query parameter based on the provided name. (default: -1)  --> JInput->get('info', -1)

Laminas\Http\Response
----------------------
Only used to get response status code texts

Laminas\Http\PhpEnvironment\Response
-------------------------------------
- getHeaders()                 | Return the header container responsible for headers. (return \Headers)
- setStatusCode()              | Set HTTP status code and (optionally) message.  (return $this)
- setContent()                 | Set message content. (return \Message object)
- sendHeaders()                | Send HTTP headers. (return $this)
- sendContent()                | Send content. (return $this)

Laminas\Http\Headers
---------------------
- addHeaderLine()              | Add a raw header line, either as separate name and value arguments, or as a single string in the form name: value (return $this)
- addHeaders()                 | Add many headers at once; expects an array of type/value pairs. (return $this)
